Note

To configure Barcode rule, select Barcode Type and give Barcode/QR Code Rule like this
"SELECT ProductCode=SUBSTRING('@SKUCode',X,Y),UDF1=SUBSTRING('@SKUCode',X,Y)".
Here X represent the starting number and Y represents the ending number.

Eg: SELECT ProductCode=SUBSTRING('@SKUCode',1,6),UDF1=SUBSTRING('@SKUCode',7,12)
